# Internware Task Manager

A modern web-based task management system designed to help teams organize tasks, track progress, and collaborate efficiently.

This project focuses on creating a clean and scalable task management interface while following modern web development practices.

---

## Project Overview

Internware Task Manager is built to simplify how tasks are created, assigned, and monitored inside a team environment.
The application provides a structured workflow that helps improve productivity and task visibility.

---

## Features

* Create and manage tasks
* Organized project structure
* Fast development environment using Vite
* Clean and responsive UI
* Modular code architecture
* Easy project setup and deployment

## Installation

Clone the repository

```
git clone https://github.com/shlvxm/Internware-Task-Manager.git
```

Move into the project folder

```
cd Internware-Task-Manager
```

Install dependencies

```
npm install
```

Run the development server

```
npm run dev
```

---

## Deployment

The project can be deployed easily using **Netlify** or any static hosting platform.

Steps:

1. Push your repository to GitHub
2. Connect the repository with Netlify
3. Deploy the project
